# 证书导出说明

## 证书信息

- **域名**: 00011000.com
- **证书类型**: Let's Encrypt SSL 证书
- **证书有效期**: 2026-01-26 至 2026-04-26
- **证书颁发机构**: Let's Encrypt (R13)

## 导出的文件

1. **certificate.pem** - 证书内容 (PEM 格式)
   - 包含完整证书链 (fullchain.pem)
   - 文件大小: 约 3.5KB
   - 位置: `/Users/xk/Documents/Workspace/aiweb/certificate.pem`

2. **private_key.pem** - 证书私钥 (PEM 格式)
   - 文件大小: 约 1.7KB
   - 位置: `/Users/xk/Documents/Workspace/aiweb/private_key.pem`

## 服务器上的原始位置

- 证书文件: `/etc/letsencrypt/live/00011000.com-0001/fullchain.pem`
- 私钥文件: `/etc/letsencrypt/live/00011000.com-0001/privkey.pem`

## 使用说明

### 在表单中填写时：

1. **证书备注名**: 可以填写 `00011000.com` 或自定义名称

2. **证书内容 (PEM 格式)**: 
   - 打开 `certificate.pem` 文件
   - 复制全部内容（包括 `-----BEGIN CERTIFICATE-----` 和 `-----END CERTIFICATE-----`）
   - 粘贴到表单的证书内容字段

3. **证书私钥 (PEM 格式)**:
   - 打开 `private_key.pem` 文件
   - 复制全部内容（包括 `-----BEGIN PRIVATE KEY-----` 和 `-----END PRIVATE KEY-----`）
   - 粘贴到表单的私钥字段

## 注意事项

⚠️ **安全提醒**:
- 私钥文件包含敏感信息，请妥善保管
- 不要将私钥文件提交到代码仓库
- 不要在不安全的环境中传输私钥
- 建议在完成导入后删除本地副本（如果需要）

## 证书续期

Let's Encrypt 证书有效期为 90 天，服务器上应该已配置自动续期。如果需要手动续期，可以运行：

```bash
ssh -i ~/.ssh/id_ed25519 root@8.140.229.172 "certbot renew"
```
